# git_HandsOn
Python scripts developed during the Git hands-on exercises.

seqClass.py:
Classifies a sequence as DNA or RNA and optionally searches for a motif.

Usage 
```bash
python3 seqClass.py -s ACTG
python3 seqClass.py -s ACTG -m TG
```

ntPerc.py:
Calculates the percentage of each nucleotide in a DNA or RNA sequence.

Usage
```bash
python3 ntPerc.py -s ACTGACTG
python3 ntPerc.py -s ACGUACGU
```
